# GlobalStyle

CSS resets and base stylesheets

```jsx
const App = () => {
  <>
    <GlobalStyle />
    <otherComponent />
  </>;
};
```

## Typography

Basic global typography

### Headings

All HTML headings, `<h1>` through `<h6>`, are available

  <Story name="Heading">
    <div>

    </div>
  </Story>
</Preview>
